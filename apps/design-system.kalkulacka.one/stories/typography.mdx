# Typography

## Fonts

### Radio Canada

[Radio-Canada](https://github.com/cbcrc/radiocanadafonts) is used as a display typeface. It's a well-internationalized set of variable fonts (weight 300–700 & width 75–100) available in normal and italic styles, made for CBC/Radio-Canada (Canada's national public broadcaster).

### Inter

[Inter](https://github.com/rsms/inter/) is used for body text. It's a well-internationalized set of variable fonts (weight 300–700) available in normal and italic styles with a lot of OpenType features, optimized for computer screens at smaller sizes, used for example by Figma.
